# Design System

## Visual Identity

- Moon jar as the central brand shape
- Hanji paper texture
- Premium Korean watercolor, gouache, and soft ink
- Warm lantern light
- Calm magical movement

## Palette

| Token | Use |
| --- | --- |
| Deep indigo | Text, night skies, navigation |
| Ivory | Background, moon jar, reading surfaces |
| Persimmon | Primary action, fruit, warmth |
| Jade | Secondary action, nature, water |
| Lotus pink | Hanbok accents, softness |
| Lantern gold | Glow, warmth, reward moments |

## Reader Layout

Default:

```text
[ full animated scene ]

[ Korean story text ]
[ optional English subtitle ]

Back   Replay/Play   Auto   Next
```

Landscape tablets may add Real Book Mode as an optional two-page layout. The default remains one large living scene per page.

## Motion Rules

- Every page may loop a subtle 3-second animation.
- No fast flashing.
- No startle audio.
- No overstimulating particle storms.
- Bedtime mode reduces brightness, tempo, and contrast.

## Asset Direction

Generated or illustrated assets should use stable character bibles. Each page should be layered when possible:

- background
- environmental motion
- characters
- light/glow
- foreground texture
