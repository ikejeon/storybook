# Shared Audio Assets

This directory is the canonical home for narration, page sounds, ambient beds, and tap-word pronunciation.

Suggested structure:

```text
audio/
  books/
    sun-and-moon/
      ko/
        page-001.m4a
      en/
        page-001.m4a
      ambient/
        forest-night-loop.m4a
  ui/
    page-flip-soft.m4a
```

Grandparent recordings should be stored locally on device or in a privacy-reviewed family storage backend, never inside the shared catalog.
